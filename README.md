# 📊 Octolab Sales Dashboard

Dashboard interativo de acompanhamento de vendas, construído com Python e Streamlit.

Exemplo prático de como transformar dados brutos em uma ferramenta de decisão visual — sem planilhas, sem processo manual.

## 🚀 Como rodar

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🔧 O que está incluído

- KPIs com variação vs período anterior
- Ranking de receita por vendedor
- Receita por produto
- Evolução semanal de vendas
- Filtros por período e vendedor
- Tabela com últimas vendas

## 🔌 Possibilidades de integração

Este dashboard roda com dados fictícios por padrão, mas pode ser conectado a:

- Banco de dados (PostgreSQL, MySQL, SQLite)
- Google Sheets
- CRM (HubSpot, Pipedrive, etc.)
- ERP
- APIs REST
